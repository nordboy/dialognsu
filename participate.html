Приглашаем студентов, аспирантов и молодых учёных

Принять участие

Конференция состоится с 27 апреля по 1 мая 2017 года в Новосибирске

Форма участия: очная.

Участие в конференции – на конкурсной основе. Оргкомитет берет на себя расходы на проживание и питание участников. Транспортные расходы – за счет отправляющей стороны.

Планируете выступать с:

  1. Докладом по научной теме
  2. Докладом о реализованном проекте в рамках тематики форума
  3. Идеей проекта (мы доработаем ее с Вами на форуме!)
  4. Хочу быть слушателем!

Ознакомьтесь с рекомендациями по оформлению тезисов и проектов.

Для участников с докладами

Предлагаем Вам ознакомиться с последними рекомендациями Министерства образования и науки РФ по подготовке и оформлению печатных публикаций

    Текст тезисов должен содержать ответы на следующие вопросы:
  1. В чем заключается актуальность темы Вашего исследования?
  2. Какова цель Вашего доклада?
  3. Какова исследовательские методы Вы применяли?
  4. Какие значимые результаты были получены?
  5. Какова практическая значимость данного исследования?

Для участников с проектами: можно пользоваться описанием по следующей схеме

  1. Социально-значимая проблема, которую решает проект.
  2. Описание особенности/новизны Вашего проекта (или подхода) в решении данной проблемы.
  3. Для кого делается проект (целевая аудитория).
  4. Ожидаемые результаты проекта (количественные и качественные).
  5. Долгосрочный (или отложенный) эффект от реализации проекта.
  6. Команда проекта, функции участников.
